# PPT to MP4 Documentation Automation

Automation workflow to convert PowerPoint content into narrated MP4 videos for product documentation and training.

## Overview
This project demonstrates a documentation automation pipeline that:
- Converts PPT slides into video
- Generates narration using Azure Text-to-Speech
- Syncs audio with slides
- Exports MP4 output for training or release notes

## Use Case
Used for enterprise SaaS documentation teams to quickly produce:
- Feature walkthrough videos
- Release highlights
- Training modules
- Customer enablement content

## Stack
- Python
- PowerPoint automation
- Azure Text-to-Speech
- Batch scripting
