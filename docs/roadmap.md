# Roadmap

## Phase 1 (Completed)
- Static PPT/PPTX to MP4 conversion baseline.
- Basic API and file-based job workflow.
- Feasibility demo and MVP validation.

## Phase 2 (Completed)
- PPT animation and transition support.
- Speaker notes to Azure TTS narration.
- End-to-end async processing with status and logs.
- Downloadable outputs through API/UI.

## Phase 3 (In Progress)
- Shared Windows-hosted service for internal teams.
- Stronger job orchestration and operational stability.
- Better progress visibility and error diagnostics.
- Configurable narration and branding options.

## Near-Term Backlog
- Authentication and access control.
- Input validation and limits for uploaded PPT files.
- Automatic cleanup policies for temporary artifacts.
- Reliability metrics (success rate, processing time).

## Future Enhancements
- Queue-based scaling for concurrent jobs.
- Optional output formats beyond MP4.
- Integration hooks for documentation/training portals.
- Enhanced template/branding presets.
